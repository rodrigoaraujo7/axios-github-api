## How to download the project:

```bash
# Clone the repository
$ git clone https://github.com/rodrigoaraujo7/axios-github-api.git

# Enter directory
$ cd axios-github-api
```

## Usage

1. Run `yarn`.<br />
2. Run `yarn dev` and access `http://127.0.0.1:5173`.<br />
